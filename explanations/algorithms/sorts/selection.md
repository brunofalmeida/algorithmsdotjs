# Selection Sort

## How it works

Selection sort takes each element and inserts it accordingly into an array.

After the algorithm iterates over every element, the final array is fully sorted.

## Time Complexity

Worst Case: O(n^2)  
Best Case: O(n^2)  

Average Case: O(n^2)
